# CrackMe Solutions

## Description

Repository ini berisi dokumentasi penyelesaian beberapa challenge CrackMe sebagai bagian dari pembelajaran mata kuliah Reverse Engineering.

## Objectives

- Melatih kemampuan Reverse Engineering.
- Menganalisis executable menggunakan Ghidra.
- Mendokumentasikan proses analisis.
- Menyusun write-up hasil challenge.

## Tools

- Ghidra
- Virtual Machine
- Windows
- x64dbg (jika diperlukan)
